# DiffusionGPT

Welcome! This is a chat interface for **DiffusionGPT**, a model that generates text using diffusion processes rather than standard autoregressive token prediction.

The code of this project can be found in the [DiffusionGPT repository](https://github.com/JorgeVanco/diffusionGPT).

### How to use
1. Enter a prompt in the chat box.
2. Watch the model "denoise" the text in real-time.
3. Adjust the **Diffusion Steps** or **Mode** in the sidebar to see how the generation changes.
